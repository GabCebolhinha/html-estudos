Modelo de Caixas

Aninhamento:uma caixa dentro de outra e de outra e assim sucessivamente.

Height=altura; width=largura. A linha ao redor da caixa é chamada de borda (border), se caso deseja criar um espaço entre a caixa e essa linha usa-se o padding. Para dar um espaço entre duas caixas (boxes) usa-se o margin. Além disso, pode-se criar outro trasejato:outline.

Existe dois tipos de caixas: box-level e inline-level.

O box-level mais usado é a tag div. o inline-level mais usado é o span. Além desses, existe para o primeiro: o nav, main, header, article, aside,footer, form, video; e para o segundo: a, code, small, strong, em, sup, sub, label, button, input, select
